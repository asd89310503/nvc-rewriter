# NVC 改寫 — iOS 捷徑食譜

目標：手機**敲背面兩下**（或動作按鈕／分享選單）→ 跳輸入框 → 打或口述 → 自動把改寫好的一句話複製到剪貼簿並顯示。

用的是你已經有的 Google Gemini 免費金鑰（直接寫在捷徑裡，只存你手機）。

---

## A. 開始前
- 準備好你的 Gemini 金鑰（在 NVC 網頁工具 ⚙️ 裡那把，或到 https://aistudio.google.com/app/apikey 領）。
- 打開內建的「**捷徑**」App → 右上「**＋**」建立新捷徑 → 右上「**ⓘ / 重新命名**」把名字改成「**好好說話**」。

---

## B. 動作一個一個加（在搜尋列打動作名稱就找得到）

> 加動作：點底部搜尋框 → 打動作名稱 → 點它，就會接在下面。

### 步驟 1：要求輸入
- 加「**要求輸入**」（Ask for Input）
- 輸入類型：**文字**
- 提示：`想講什麼？`

### 步驟 2：把「提示詞 + 你的話」合成一段文字
- 加「**文字**」（Text）
- 內容貼上下面這段，**最後面**接上步驟 1 的變數「提供的輸入」：

```
你是溝通教練，專長非暴力溝通(NVC)。下面是我原本想講、但語氣不好的一句話。
請改寫成一句自然口語、台灣用法的繁體中文：說出我的觀察/感受/需要/請求，
不指責、不命令、不評價對方，但也不要委婉到失去重點。
只輸出改寫後那一句話，不要任何說明、引號或前後綴。嚴禁簡體字。

原句：
```
> 貼完後，游標停在「原句：」後面，點鍵盤上方的變數列，插入「**提供的輸入**」。

### 步驟 3：組請求內容（字典，最關鍵一步）
- 加「**字典**」（Dictionary）
- 照這個樹狀結構建（每次點「新增項目」會問你「類型」和「鍵」）：

```
字典
└─ 鍵 contents     類型：陣列(Array)
    └─ 項目1        類型：字典(Dictionary)
         └─ 鍵 parts   類型：陣列(Array)
              └─ 項目1   類型：字典(Dictionary)
                   └─ 鍵 text   類型：文字  值＝步驟2的「文字」變數
```

> 怎麼點：
> 1. 新增項目 → 類型選**陣列** → 鍵打 `contents` → 進去再「新增項目」→ 類型**字典** → 進去新增 `parts`（**陣列**）→ 進去**字典** → 新增 `text`（**文字**），值的地方插入步驟 2 的「文字」變數。
>
> 註：`generationConfig / temperature` 可省略（Gemini 會用預設值，品質一樣好）。若手機字典的「數字」欄位無法輸入 0.7，直接不要這段即可。

### 步驟 4：呼叫 Gemini
- 加「**取得 URL 內容**」（Get Contents of URL）
- URL 填（把 `你的金鑰` 換成你的金鑰）：
```
https://generativelanguage.googleapis.com/v1beta/models/gemini-2.5-flash:generateContent?key=你的金鑰
```
- 點「顯示更多」展開：
  - 方法（Method）：**POST**
  - 標頭（Headers）：新增一個 → 鍵 `Content-Type`、值 `application/json`
  - 請求內文（Request Body）：選 **JSON** → 選「**字典**」（步驟 3 那個變數）

### 步驟 5：從回應挖出改寫結果
依序加這幾個動作（這段在挖 JSON，照順序加就好）：
1. 「**取得字典值**」：取得 `candidates` 的值（來源是步驟 4 的 URL 內容）
2. 「**從列表中取得項目**」：取「第一個項目」
3. 「**取得字典值**」：取得 `content` 的值
4. 「**取得字典值**」：取得 `parts` 的值
5. 「**從列表中取得項目**」：取「第一個項目」
6. 「**取得字典值**」：取得 `text` 的值 ← 這就是改寫好的句子

### 步驟 6：複製 + 顯示
- 加「**拷貝到剪貼板**」（Copy to Clipboard）→ 對象是步驟 5 最後的「字典值」
- 加「**顯示結果**」（Show Result）→ 內容同上（讓你看到改寫，且已經複製好，貼上即可用）

## C. 設定「隨時隨地一鍵叫出來」（重點）

捷徑建好後，到「設定」App 綁定觸發方式，挑你喜歡的：

- **敲背面兩下** ⭐最推：設定 → 輔助使用 → 觸控 → 背面輕點 → 輕點兩下 → 選「好好說話」。之後任何畫面敲手機背面兩下就跳出來。
- **動作按鈕**（iPhone 15 Pro 以上）：設定 → 動作按鈕 → 選捷徑 → 好好說話。
- **加到主畫面**：捷徑 App 裡點捷徑的「分享」→ 加入主畫面，桌面多一顆圖示。
- **分享選單**：在任何 App 選取文字 → 分享 → 好好說話（適合改寫別人傳來的訊息回覆）。需在捷徑設定裡開啟「在分享表單中顯示」並接受文字輸入。
- **Siri**：直接喊「嘿 Siri，好好說話」。

---

## D. 測試與排錯
- 第一次跑會問你權限（允許傳送資料到 Google）→ 允許。
- 沒反應／報錯：
  - 出現金鑰相關錯誤 → 檢查步驟 4 的 URL 金鑰有沒有貼對、有沒有多空格。
  - 顯示一堆英文 JSON → 步驟 5 的挖值順序錯了，對照樹狀檢查 `candidates → 第一項 → content → parts → 第一項 → text`。
  - 想換語氣（更直接／更溫和）→ 改步驟 2 那段提示詞文字即可。
  - 想換模型 → 改步驟 4 URL 裡的 `gemini-2.5-flash`。
